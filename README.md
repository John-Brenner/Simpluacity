**SimpLuacity**

A collection of Lua scripts created or modified for Final Fantasy XI, created to prioritize readability, consistency, and ease of use for new users.

**---Key Features---**

 **Readble Design:** Scripts are intentionally simplified with consistent formatting and minimal abstration.

**Template-Based Structure:** Includes reusable base scripts that allow users to implement GearSwap by only editing gear tables.

**Self-Contained Utilities:** Complex exceptions (such as Bars and custom situation gear logic) include built-in help systems to ensure usability without external references.
